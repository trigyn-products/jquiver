## Versioning Notes
This repository currently uses a simple documentation changelog. Future versions may adopt semantic versioning after ownership and release process are finalized.

Suggested version meaning:
- Major: Large restructure or incompatible agent behavior changes.
- Minor: New JQuiver modules, playbooks, examples, or reference pages.
- Patch: Clarifications, TODO updates, typo fixes, and small verified facts.

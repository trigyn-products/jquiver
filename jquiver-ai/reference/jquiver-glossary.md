# JQuiver Glossary

## Purpose
Define common JQuiver terms for agents and developers.

## When to use this file
Use this file when reading documentation or explaining modules.

## Related files
- `metadata-table-reference.md`
- `../KB-MAP.md`

## Known facts
- Route: metadata entry mapping a URL to a target.
- Dynamic form: metadata-backed form configured in `jq_dynamic_form`.
- Grid: metadata-backed listing configured in `jq_grid_details`.
- Dynamic REST: metadata-backed API configured in `jq_dynamic_rest_details`.
- Additional datasource: metadata-backed connection to another database.
